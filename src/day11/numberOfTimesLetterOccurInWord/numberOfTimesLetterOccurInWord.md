# Number Of Times Letter Occur In Word

### Challenge

When passed a letter and an array of letters, numberOfTimesLetterOccurInWord() should return the letter with the most number of cocurances within that list of letters   

### Sample Test Cases

Input: ("l", "hello")
Output: 2

Input: ("a", "what")
Output: 1

Input: ("b", "what")
Output: 0
